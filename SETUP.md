# セットアップ手順

## いちばん簡単な使い方

1. ZIPを解凍
2. `index.html` をブラウザで開く
3. 診断フォームに入力
4. 結果が表示される

## ファイル構成

```
oshikatsu-complete-package/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
├── assets/
│   └── mockup images
└── docs/
    ├── UI_DESIGN.md
    ├── FLOW_MAP.md
    ├── FEATURES.md
    └── SETUP.md
```
