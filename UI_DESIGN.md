# UIデザイン仕様書

## サイト名
推し活資金AI

## コンセプト
節約するための家計簿ではなく、推し活・ライブ遠征・旅行・美容のためにお金を作る無料ブラウザ診断サイト。

## デザイン方向性
- 明るい
- 女性向け
- パステル
- 家計管理の安心感
- 推し活のワクワク感
- カード型UI
- スマホでも使いやすい診断導線

## フォント
- 基本：Noto Sans JP / Hiragino Sans / Yu Gothic
- 見出し：太字、やや丸みのある印象
- 数字：大きく、ピンクで強調

## 主要UIパーツ
### ヘッダー
- 左：ロゴ
- 中央：ナビゲーション
- 右：無料診断CTA

### ヒーロー
- 大見出し「推しを諦めない。」
- サブコピー
- 4つの診断カード
- 右側にサンプル診断結果

### 診断カード
- 角丸20px
- 薄い影
- アイコン
- 番号ラベル
- CTAボタン

### 結果画面
- 大きな金額表示
- 年間予算
- 遠征可能回数
- 削減可能額
- 注意文言

### 注意文言
診断結果は参考情報です。契約・予約・購入などの最終判断は利用者自身で行ってください。
