# 機能一覧

## MVP機能

| 機能 | 内容 | 入力 | 出力 |
|---|---|---|---|
| 推し活資金診断 | 収入と支出から推し活予算を計算 | 手取り、家賃、食費、通信費、光熱費、サブスク、その他、貯金 | 月間推し活予算、年間予算、遠征可能回数 |
| 娯楽費捻出診断 | 支出から削減候補を計算 | 通信費、サブスク、食費、コンビニ代 | 月間削減可能額、年間削減可能額、優先順位 |
| 遠征プランナー | 遠征費の目安を計算 | 出発地、目的地、宿泊数、食費 | 最安・バランス・快適プラン |
| 目標シミュレーター | 目標達成までの期間を計算 | 目標名、必要金額、毎月積立額 | 達成月数、年間積立額 |

## 第二段階機能

| 機能 | 内容 |
|---|---|
| AI家計診断 | 支出をカテゴリ分けし、改善候補を提示 |
| レシート読取 | 画像から購入品・金額を抽出 |
| 食費診断 | コンビニ率・外食率を分析し、献立と買い物リストを生成 |
| 通信費診断 | 利用GBと月額料金から見直し候補を提示 |
| 副業適性診断 | 稼働時間・スキルから副業候補を提示 |

## 継続利用機能

| 機能 | 内容 |
|---|---|
| 推し活家計簿 | グッズ代、チケット代、遠征費を記録 |
| 推し別管理 | 複数の推しごとに支出管理 |
| 年間レポート | 年間推し活総額・遠征回数・削減額を表示 |
| 会員登録 | 診断結果の保存 |
| プレミアム | レシート保存、年間計画、複数目標管理 |

## 免責方針
- AIや診断は参考情報として表示
- 契約・予約・購入の最終判断は利用者に委ねる
- 通信費・旅行費など変動する情報は公式サイト確認を促す
